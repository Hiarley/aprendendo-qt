---------------------------------
Descrição dos programas
Autores: Luan, Danilo
---------------------------------

- Teste2
Calculadora utilizando interface gráfica.

- Janelas
Programa que possui duas janelas, sendo uma QDialog. Existe a comunicação entre as duas por signals e slots.

- Teste5
Programa que vizualiza imagens do computador, inverte cores, salva a imagem, aplica efeitos. 

- Imagem_opencv
Programa que utiliza as funções básicas do OpenCV.

- SuperMercado
Programa que utiliza SQLite, compras de produtos.

- Calculadora
Programa ANDROID, feito no eclipse(JAVA), é uma calculadora comum com função de somar.

- Calculadora2
Programa ANDROID, feito no eclipse(JAVA), finalizando a calculadora, com função de somar, subtrair, multiplicar e dividir. 

- Feira
Programa Android que simula compras em uma feira, utilizando Chackbox, radiobutton, alert, evento.

- Janelas_android
Programa Android que faz passagem de janelas, há interações entre janelas.

- Imagens
Programa Android, utiliza botões com imagens, trocas de janelas.

- Galeria
Programa Android, visualiza uma galeria de imagens

- Menus
Programa Android, visualiza o Menu do programa.

- Cadastro
Programa Android, Utiliza o Banco de dados, Insere e Visualiza os Dados.

- Stability_systems
Programa para o projeto de detecção de estabilidade de sistemas no Android.

- Aplicacao_OpenGL
Programa em QT que utiliza OpenGL na criação de imagens

- My_week
Programa para o projeto My Week para Android.
